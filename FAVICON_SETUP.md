# Favicon Setup

Bitte erstelle die folgenden Favicon-Dateien und platziere sie im `public/` Ordner:

## Benötigte Dateien:

1. **favicon.ico** (16x16 und 32x32)
2. **favicon-16x16.png** (16x16)
3. **favicon-32x32.png** (32x32)
4. **apple-touch-icon.png** (180x180)
5. **android-chrome-192x192.png** (192x192)
6. **android-chrome-512x512.png** (512x512)

## Design:

Das Icon sollte das gleiche Target/Zielscheiben-Design wie pathly.com haben:
- Drei konzentrische Kreise
- Minimalistisch schwarz/weiß
- Transparenter Hintergrund

## Tool zum Erstellen:

Du kannst diese Website verwenden: https://realfavicongenerator.net/

1. Lade ein SVG oder PNG mit dem Design hoch
2. Generiere alle benötigten Formate
3. Lade das Paket herunter
4. Kopiere alle Dateien in den `public/` Ordner

Alle Referenzen sind bereits in `index.html` und `site.webmanifest` eingebunden!
